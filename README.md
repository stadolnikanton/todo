# To-Do Web Application

Веб-приложение для управления задачами с регистрацией пользователей.
Проект написан на Python с использованием MVC-паттерна.

## Стек технологий
* Python 3.12+
* Пакетный менеджер: uv
* База данных: SQLAlchemy + Alembic
* Шаблонизатор: Jinja2 (HTML/CSS)

## Установка и запуск

1. Установите зависимости через uv:
   uv sync

2. Примените миграции базы данных:
   alembic upgrade head

3. Запустите сервер:
   python todo/main.py

## Структура
* /controller - Логика обработки запросов
* /model - Модели базы данных
* /view - Представления
* /templates - HTML файлы
